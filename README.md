# Full Stack Portfolio

## Description
This is a modern, responsive full-stack developer portfolio built with React on the frontend and a simple backend (Node.js/Express or mock APIs) to demonstrate dynamic data handling. It showcases my skills, projects, resume, and contact form â€” all in one place.

## Features
- Responsive UI across all screen sizes
- Projects section with live links and GitHub integration
- About Me and Skills section
- Contact form (can integrate with backend or service like Formspree)
- Resume download button

## Tech Stack
- **Frontend:** React.js, HTML, CSS, JavaScript
- **Backend (optional):** Node.js, Express.js
- **Styling:** Tailwind 
- **Deployment:** GitHub Pages / Vercel / Netlify

## Getting Started

### 1. Clone the repo
git clone https://github.com/archiporwal00/fullstack-portfolio.git

### 2. Install dependencies
cd fullstack-portfolio
npm install

### 3. Run the app locally
npm start
